TinX
=====

"T in X" - A simple time constants library.

## Installation

    $ npm install tinx
    
## Use

    var t = require('tinx');
    
    console.log(t.milliseconds.in.a.second); // 1000
    console.log(t.seconds.in.an.hour); // 3600
    console.log(t.hours.in.a.day); // 24
